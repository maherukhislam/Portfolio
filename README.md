# Maherukh Islam Portfolio

React + Vite portfolio site prepared for free GitHub Pages hosting.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## GitHub Pages

1. Push this repository to GitHub on the `main` branch.
2. In GitHub, open `Settings -> Pages`.
3. Set `Source` to `GitHub Actions`.
4. Push to `main` again if needed.

The workflow in `.github/workflows/deploy.yml` will build and deploy automatically.
